# Yuzu Emulator

> **Yuzu** — Popular Nintendo Switch emulator that allows you to play Switch games on your PC.

[![Stars](https://img.shields.io/github/stars/pedrodg28/yuzu-emu)](https://github.com/pedrodg28/pedrodg28/yuzu-emu) [![License: MIT](https://img.shields.io/badge/License-MIT-red.svg)](https://opensource.org/licenses/MIT)

## Overview
**Yuzu Emulator** is a highly optimized emulator for Nintendo Switch games. It enables you to play your favorite Switch titles on Windows PC with improved performance, higher resolutions, widescreen support, and modern features.

> [!Important]
> This emulator is for educational and game preservation purposes only. You must legally own the original game files (dumps) to use them with this emulator.

## Key Features
- High compatibility with Nintendo Switch games
- Improved performance and stability
- Higher resolution and widescreen support
- Keyboard and full controller support
- Enhanced graphics and rendering options
- Save states and cheat support
- Lightweight and fast loading

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `YuzuEmulator.exe`
3. Add your legally dumped Nintendo Switch game files
4. Configure controls and graphics settings
5. Start playing

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Emulator doesn't start     | Run as Administrator                          |
| Low FPS                    | Lower resolution or graphics settings         |
| Game crashes               | Update GPU drivers and verify game files      |
| Antivirus warning          | Add the folder to exclusions                  |
| Black screen               | Switch between OpenGL and Vulkan backend      |
